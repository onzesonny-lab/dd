# Code Runner App

A simple web-based JavaScript code runner where you can write code and make it happen by running it.

## Usage

1. Open `index.html` in your web browser.
2. Write your JavaScript code in the textarea.
3. Click the "Run Code" button to execute the code.
4. View the results and console output in the output area below.

## Features

- Free and fast JavaScript code execution.
- Supports numbers, text, and any valid JS code.
- Displays results and console.log outputs.
- Error handling for invalid code.

## Example

Try this code:
```javascript
console.log('Hello, World!');
let sum = 5 + 10;
sum;
```

This will output "Hello, World!" in the console and return 15 as the result.
